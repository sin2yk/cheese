## ⑦フリー項目（感想、シェアしたいこと等なんでも）
- A11y配慮：
  - フォームは「項目名（ラベル）を押すと入力欄にフォーカスが移る」つくり。
    注意書きもその入力欄とセットで読まれるので、迷いません。
  - 画面に書いてある言葉をそのまま読み上げる方針にし、隠しラベルの固定文は最小限。
    （“動的なのに中身は固定”にならないように配慮）。ただし、「ACCESS」「CONTACT」など
    アルファベット見出しは日本語で読み上げるようにしました。
  -リンクは「マウスを乗せた時」だけでなく「キーボードで選択した時（Tab移動）」も
    色や下線がくっきり変わり、フォーカス枠も表示されます（見失わない工夫）。
  - キーボードだけでも送信まで進めます。Tabで移動でき、今どこにいるか枠で分かります。
  - 外部リンクは新しいタブで開き、安全設定（noopener）で意図しない遷移を防止。
